Tontapatt est une application web de mise en relation de client souhaitant faire entretenire son terrain et d'éleveur proposant un service d'écôpaturage.

## Installation
---
Tontapatte est une application .net core traditionel

![css version](https://img.shields.io/badge/.NET-5C2D91?style=for-the-badge&logo=.net&logoColor=white)
- Cloner le dépot directement dans visual studio
- Demarrer la solution

![css version](https://img.shields.io/badge/MySQL-00000F?style=for-the-badge&logo=mysql&logoColor=white)
- Creez une BDD `tontapatt_bdd`
- Ouvrir le script `~\Source\Repos\Tontapatt_Groupe_2\DataAccess\bdd\base_v18.sql`
- Executer

Vous êtes prêt a demarrer votre serveur IIS Express
